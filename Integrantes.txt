1. Investigar y escribir una breve descripción sobre lo que es un RDBMS.

Un Sistema de Gestión de Bases de Datos Relacionales (RDBMS, por sus siglas en inglés) es un software diseñado para gestionar bases de datos que almacenan datos en forma de tablas, las cuales están relacionadas entre sí a través de claves primarias y foráneas. Los RDBMS utilizan el lenguaje SQL (Structured Query Language) para realizar operaciones sobre los datos, como insertar, actualizar, eliminar y consultar información.
